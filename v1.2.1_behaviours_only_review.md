# v1.2.1 behaviour-pack audit

Concise status of `v1.2.1_src_only/behaviors` against the current `BOB_behaviors` after the direct ports and current-branch adaptations.

## Audit method

- Compared every v1.2.1 behaviour file with `BOB_behaviors/<same-relative-path>` byte-for-byte.
- Checked the reverse direction for files that exist only in current main.
- Treated current-branch API/performance fixes as authoritative where a blind v1.2.1 copy would regress them.
