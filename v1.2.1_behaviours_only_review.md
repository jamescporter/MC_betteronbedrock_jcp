# v1.2.1 remaining behaviour-pack review

This file now lists only v1.2.1 behaviour work that still needs a decision or review. Completed direct ports and completed current-branch adaptations are intentionally omitted.

## Audit basis

- Compared `v1.2.1_src_only/behaviors` with `BOB_behaviors` by exact relative path and byte content.
- Current byte counts after completed ports: **3073** identical, **107** changed same-path, **0** unresolved v1.2.1 source-only, **3** resolved v1.2.1 source-only, **21** current-main-only.
- Of the changed same-path files, **56** still need review. The item-behaviour script set covering custom spear, ghost necklace, staffs, void boots, void totem, food effects, and custom tool durability is now code-audited and resolved. The quest/WAWLA script set is also code-audited; in-game validation is still required before calling that gameplay behaviour 100% verified. The other changed files are completed adaptations or intentionally retained current-branch differences.

## Enchantment and strip-block validation status

The enchantment/strip-block script review is code-complete against v1.2.1 and current-main expectations, but it is **not** marked as 100% gameplay-validated because automated checks cannot exercise Bedrock block handles, protected-block behaviour, chunk availability, item durability mutation, spawned drops/XP, or block permutation writes.
